Underneath the packaging is the quick start user guide.

AYA NEO user guide

There is a USB Type-C cable for charging your AYA NEO.

AYA NEO charge cable

There are two [USB Type-C](https://en.wikipedia.org/wiki/USB-C) to Type A adaptors which can be plugged into the AYA NEO gaming handheld PC. With these you can connect peripherals such as a mouse, keyboard or memory stick.

Useful Type C to Type A USB adaptors

And a 60W [PD charger](https://blog.ravpower.com/2018/02/all-about-usb-power-delivery/) is now included. It comes with plug adaptors for UK, Europe and Southeast Asia. Wherever you are, you should be able to easily charge up your AYA NEO! It is a good addition by [Aya](https://droix.net/blogs/brand/aya/) as it now comes with a good charger and adaptors for the main countries.
