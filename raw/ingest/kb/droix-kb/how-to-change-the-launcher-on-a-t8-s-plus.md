If you find that you’ve accidentally set the wrong launcher on your DroiX® T8-S Plus device, this post shows how to switch back  
  
Below we have a screenshot of the launcher most people use with their DroiX® T8-S Plus.

This is the second launcher installed on your T8-S Plus.

When you see this sort of option, until you’re certain which launcher you’d like to use, select the Just Once option.

Until you select the Always option, each time you press the home button on your remote control, you’ll be asked to confirm which one to use.

If you have installed a 3rd party launcher, the following still applies, however you’ll see an extra entry in the list of available launchers.

#### How To Change The Launcher

First, click the Settings icon from your launcher’s homescreen (both launchers have this shortcut). If you are using a 3rd party launcher and can’t see an icon of a cog, locate and open the list of all installed applications, then click Settings.

From here, we need to go to “More setting”.

Now we need the Home (in the Device section) entry, please click it.

Finally, select the launcher you want to use instead of the current one, and return to the previous screen. Now press the home button to test everything has turned out well.
