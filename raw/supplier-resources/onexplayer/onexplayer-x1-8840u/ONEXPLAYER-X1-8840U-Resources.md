Flashing Method - Desktop Flash: 1. Plug in the charger. 2. Disable Microsoft's built-in Virus & Threat Protection. 3. Extract the files to the desktop (remove any Chinese characters from the extracted folder name). 4. Select the "exe" file in the folder and right-click select "Run as administrator." 5. Once the flashing is complete, you should see a message saying "successfully" or "completed." The system will then automatically restart. Important: After the BIOS flashing is complete, the system will take approximately 2-3 minutes to match the underlying settings during the first boot. Please be patient and do not force shut down the computer.
