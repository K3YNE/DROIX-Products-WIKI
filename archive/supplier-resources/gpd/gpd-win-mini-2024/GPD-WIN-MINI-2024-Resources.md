15. Fixed a issue that could cause a freeze when setting TDP consecutively.
16. Fixed an issue where certain options in specific game profiles, such as CPU frequency limits, were not applied automatically.
17. Fixed an issue where frame locking failed in some specific game profiles. Changed the frame locking strategy for specific profiles; it no longer sets a global configuration but instead uses an RTSS process-specific configuration for frame locking.
18. Fixed an issue causing the application to crash when refreshing the tray icon.

Methods for Updating AMD Graphics Card Drivers:
The official driver package may not be updated in real time. After a period of device usage, the driver version may become outdated, potentially involving multiple bugs. This requires manual upgrades to the latest drivers. Below are several primary methods for updating hardware drivers:
Open the AMD official driver download page: https://www.amd.com/en/support/download/drivers.html
Processor Keywords for Input Devices: Select the corresponding processor, then click Search
On the opened page, expand the drivers corresponding to your pre-installed Windows system version, locate the driver with the "WHQL Recommended" label, and download and install it.
AMD Chipset Driver Update Methods:
Open the AMD official driver download page: https://www.amd.com/en/support/download/drivers.html
Follow the steps in the image below in order, then click ‌"Submit"‌.
On the opened page, expand the drivers corresponding to your pre-installed Windows system version and download/install them. If you find the version unstable, you can click ‌"Previous Version"‌ below to download a more stable release.
Intel Driver Update Methods:
For GPD devices with AMD processors, the only hardware involving Intel drivers is the Wi-Fi and Bluetooth chip. Open the page: https://www.intel.com/content/www/us/en/support/detect.html
Download and install ‌Intel® Driver & Support Assistant (Intel® DSA)‌. It is not a driver itself but can identify the driver versions of Intel hardware in your device. If the version is outdated, it will automatically connect to the internet for an upgrade.
