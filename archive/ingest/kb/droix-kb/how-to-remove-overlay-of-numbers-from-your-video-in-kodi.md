If you find you have text on top of your videos in Kodi, SPMC, XBMC or DBMC like this:

Then debug logging has been turned on. To turn it off, return to Kodi’s homescreen, scroll to and click ont he System menu. Depending on the skin you’re using right now, it may be titled Settings. In either case, click this menu entry directly, then on to the System menu (just below the Services menu). Now select the Debugging entry and make sure Enable Debug Logging isn’t turned on. The image below shows this option enabled, which is why you can still see the stats overlay.

TL;DR?  
Open Kodi/DBMC/SPMC/XBMC>System (Settings if no System menu)>System>Debugging>Enable Debug Logging TURN THIS OFF
