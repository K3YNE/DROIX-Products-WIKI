# GPD WIN 5 BIOS v2.25

## Version Notes

AMD has officially updated the BIOS code. This is a completely new version. **After upgrading, it is not possible to flash back to older BIOS versions.**

## Installation

In Windows, after downloading and extracting the files, double-click to execute the upgrade. The system will automatically restart and enter the BIOS interface to perform the upgrade. After a successful upgrade, the system shuts down automatically; a manual restart is required.

**Note:** During the upgrade, ensure the device is connected to a power source.
