# Qualcomm Snapdragon 8+ Gen 1

The Snapdragon 8+ Gen 1 is a Qualcomm mobile SoC used in the [[ayaneo-pocket-vert|AYANEO Pocket VERT]]. The [[ayaneo-pocket-vert-product-page]] lists an 8-core CPU up to 3.20 GHz, Adreno 730 GPU up to 800 MHz, and 10-12W TDP.

## Key Specs

| Spec | Value |
|---|---|
| CPU | 8 cores, up to 3.20 GHz |
| GPU | Qualcomm Adreno 730 |
| GPU frequency | Up to 800 MHz |
| TDP in tracked product | 10-12W |
| Memory support in tracked product | LPDDR5X at 6400 MT/s |
