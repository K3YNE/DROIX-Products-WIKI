# Intel Processor N250

The Intel Processor N250 is an entry-tier N-series (Alder Lake-N) chip built on Intel's 7 process node (10nm ESF). It is designed for fanless or near-silent thermal envelopes, suited to field-work UMPCs rather than gaming. In the DROIX lineup it powers the entry variant of the [[gpd-micropc-2|GPD MicroPC 2]].

## Key Specs

| Spec | Value |
|------|-------|
| Cores / Threads | 4C / 4T (all efficiency cores) |
| Base / Boost | 1.0 GHz / 3.3 GHz |
| GPU | Intel UHD Graphics (32 EU) |
| TDP | 6W |
| Process | Intel 7 (10nm ESF) |

## Products Using This Chip

- [[gpd-micropc-2]] — ruggedised UMPC for IT/field work; entry variant: [[gpd-micropc-2-n250]]

## Performance Context

The N250 provides adequate performance for productivity, web, and light Linux/Windows workloads in a low-power envelope. For multi-threaded field-work tasks (e.g. network diagnostics, data processing), the [[intel-n300|N300]] variant of the [[gpd-micropc-2]] offers meaningfully higher throughput. Not suited to gaming. See [[umpc]] for category context.
