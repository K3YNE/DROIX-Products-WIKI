# Intel Processor N300

The Intel Processor N300 is a mid-tier N-series (Alder Lake-N) chip built on Intel's 7 process node (10nm ESF). It offers significantly higher multi-threaded throughput than the [[intel-n250|N250]] while maintaining a low-power envelope. In the DROIX lineup it powers the higher variant of the [[gpd-micropc-2|GPD MicroPC 2]].

## Key Specs

| Spec | Value |
|------|-------|
| Cores / Threads | 8C / 8T (all efficiency cores) |
| Base / Boost | 0.9 GHz / 3.8 GHz |
| GPU | Intel UHD Graphics (32 EU) |
| TDP | 6W |
| Process | Intel 7 (10nm ESF) |

## Products Using This Chip

- [[gpd-micropc-2]] — ruggedised UMPC for IT/field work; higher variant: [[gpd-micropc-2-n300]]

## Performance Context

The N300 doubles the core count of the [[intel-n250|N250]], making a material difference for multi-threaded field-work tasks. The DROIX review of the [[gpd-micropc-2]] highlighted this as the key differentiator between the two SKUs. See [[umpc]] for category context.
