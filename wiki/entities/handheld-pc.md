# Handheld PC

Windows-based handheld gaming PCs. These run full Windows and are designed for gaming, emulation, and general-purpose PC use in a handheld form factor. They offer the broadest software compatibility of any handheld category but require more power and generate more heat than Android handhelds.

## Key Buying Considerations

- **TDP / performance**: Strix Halo (Ryzen AI Max) is the current performance tier for handheld PCs. See [[amd-ryzen-ai-max]] and [[windows-handheld-gaming]].
- **Size / weight**: Larger and heavier than Android handhelds — typically 550g–750g before external battery.
- **Battery life**: Windows is less power-efficient than Android; external battery systems are common.
- **Form factor**: [[gpd-win-5]] is 7", [[onexplayer-onexfly-apex]] is 8", [[ayaneo-flip-1s-ds]] is a dual-screen clamshell.
- **Storage**: Mini SSD slot is a newer format; full M.2 support on some models.
