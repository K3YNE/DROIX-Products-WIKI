# GPD

GPD (GamePad Digital) manufactures Windows handheld PCs and ultra-mobile PCs (UMPCs).

## Key Products Reviewed

- [[GPD WIN 5]] - Strix Halo flagship handheld (first Strix Halo device to market)
- [[GPD MicroPC 2]] - UMPC designed for field work and IT professionals
- [[GPD WIN 5 Smart Dock]] - docking accessory (poorly received)
- **GPD Win 4** - predecessor handheld with integrated keyboard
- **GPD G1** - external GPU dock
- **GPD Pocket 4** - ultra-mobile PC

## Hardware Characteristics

- The Win series historically featured integrated keyboards, though the [[GPD WIN 5]] dropped this feature
- External battery systems for extended portable use
- Mini SSD format adoption for internal storage
- Fingerprint sensors on select models

## Notable Achievements

The [[GPD WIN 5]] was the first Strix Halo handheld to reach reviewers and set early performance benchmarks for the platform, competing directly with the [[ONEXFLY Apex]] which uses the same processor.
