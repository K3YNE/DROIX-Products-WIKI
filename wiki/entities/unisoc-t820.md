# Unisoc T820

The Unisoc T820 is a mid-range Android SoC built on a 6nm process. It targets budget handhelds focused on retro gaming and emulation, offering reasonable performance at a low cost. In the DROIX lineup it powers the [[anbernic-rg-slide|Anbernic RG Slide]].

## Key Specs

| Spec | Value |
|------|-------|
| CPU | 4× Cortex-A75 + 4× Cortex-A55 |
| GPU | Mali-G57 MP4 |
| Process | 6nm |

## Products Using This Chip

- [[anbernic-rg-slide]] — budget sliding-screen Android handheld

## Performance Context

The T820 delivers performance suitable for retro gaming up to PS2/PSP generation and lighter emulation. It sits in the budget tier below Snapdragon-based handhelds. The [[anbernic-rg-slide]] review noted it as adequate for the target retro-gaming audience but not suited to demanding emulators like Vita3K or PS3. See [[emulation-performance]] for tier context.
