# Qualcomm Snapdragon 8 Elite

The Snapdragon 8 Elite is Qualcomm's 2024 flagship mobile SoC, built on a 3nm TSMC process. It is the first Snapdragon to use Qualcomm's custom Oryon CPU cores (replacing ARM Cortex designs). In the DROIX lineup it powers the [[ayn-odin-3|AYN Odin 3]], which held the Android benchmark champion title upon its launch.

## Key Specs

| Spec | Value |
|------|-------|
| CPU | 2× Oryon (3.53 GHz) + 6× Oryon (3.33 GHz) |
| GPU | Adreno 830 |
| Process | 3nm TSMC |
| NPU | Hexagon |

## Products Using This Chip

- [[ayn-odin-3]] — flagship Android handheld; Android benchmark champion at time of review

## Performance Context

At launch in early 2026, the Snapdragon 8 Elite-powered [[ayn-odin-3]] became the Android performance champion across DROIX reviews, displacing the [[qualcomm-snapdragon-g3-gen-3|G3 Gen 3]]-powered [[ayaneo-pocket-s2|Pocket S2]]. See [[android-handheld-gaming]] for the broader context.
