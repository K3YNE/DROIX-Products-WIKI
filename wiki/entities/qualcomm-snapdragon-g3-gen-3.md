# Qualcomm Snapdragon G3 Gen 3

Qualcomm's Gen-3 Snapdragon G-series gaming SoC. Targets premium Android gaming handhelds.

## Key Specs

| Spec | Value |
|---|---|
| CPU | 8 cores, up to 3.36 GHz |
| CPU architecture in Pocket S2 source | 1+5+2 with Kryo Prime core at 3.3 GHz |
| GPU | Qualcomm Adreno A33 |
| GPU frequency | Up to 1050 MHz |
| TDP in tracked Pocket S2 source | 20W |
