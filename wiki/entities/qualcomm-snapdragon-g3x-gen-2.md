# Qualcomm Snapdragon G3x Gen 2

The Snapdragon G3x Gen 2 is a Qualcomm gaming handheld SoC used across the [[ayaneo|AYANEO]] Pocket lineup: [[ayaneo-pocket-ace|Pocket ACE]], [[ayaneo-pocket-dmg|Pocket DMG]], [[ayaneo-pocket-ds|Pocket DS]], [[ayaneo-pocket-evo|Pocket EVO]], and [[ayaneo-pocket-s|Pocket S]]. The [[ayaneo-pocket-ace-product-page]] and [[ayaneo-pocket-ds-product-page]] list an 8-core CPU up to 3.36 GHz, Adreno A32 GPU up to 1000 MHz, and up to 15W sustained performance / TDP.

## Key Specs

| Spec | Value |
|---|---|
| CPU | 8 cores, up to 3.36 GHz |
| GPU | Qualcomm Adreno A32 |
| GPU frequency | Up to 1000 MHz |
| Sustained performance claim | Up to 15W |
| Memory support in tracked product | LPDDR5X at 8533 MT/s |

- [[ayaneo-pocket-dmg]] — vertical Game Boy-style [[android-handheld]].
- [[ayaneo-pocket-ds]] — clamshell dual-screen [[android-handheld]].
- [[ayaneo-pocket-evo]] — 7" OLED [[android-handheld]].
- [[ayaneo-pocket-s]] — 6" IPS [[android-handheld]].
