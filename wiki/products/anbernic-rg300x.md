# Anbernic RG300X

Legacy retro gaming handheld from [[anbernic|Anbernic]]. Ingenic JZ4770 dual-core @ 1.0 GHz, Vivante GC860 GPU. 3" IPS, 640x480, 4:3. Twin SD card design. Mini HDMI, dual speakers, built-in rumble. No Wi-Fi or Bluetooth. Capable up to PS1 emulation.

## Specs

| Spec | Value |
|------|-------|
| CPU | Ingenic JZ4770 dual-core @ 1.0 GHz |
| GPU | Vivante GC860 @ 575 MHz |
| RAM | 512 MB LPDDR2 |
| Storage | 64 GB MicroSD + 1x expansion slot |
| Display | 3" IPS, 640x480, 266 PPI, no touchscreen |
| Battery | 2500 mAh Li-Ion (~5 hours) |
| Connectivity | None |
| Ports | USB-C, 3.5mm jack, Mini HDMI |
| OS | Linux |
