# ONEXPLAYER X1 Mini

8.8" 3-in-1 Windows handheld from [[onexplayer|ONEXPLAYER]] powered by AMD Ryzen 7 8840U with Radeon 780M. Features a 2.5K 144Hz LTPS display, detachable controllers for multiple modes (tablet, console, laptop, tabletop, TV), swappable M.2 2230/2242 SSD, native OCuLink, and 65.02 Wh battery with 100W GaN charger. Weighs ~710g (main body). Out of stock at DROIX.

## Design & Build

- 8.8" LTPS 2.5K display, ~710g (main body only)
- Two-thirds the size of an A4 sheet
- 3-in-1+ design: tablet, console, laptop, tabletop, TV modes
- Detachable controllers with ergonomic curvature design
- Hall effect joysticks with tactile triggers
- Fingerprint sensor (Windows Hello)
- Active cooling: 4700 RPM fan with hydraulic bearing
- Harman AudioEFX dual speakers
- OneXConsole game software (Super Game Library, Performance Control Center, Game Cloud Storage)
- Windows 11 Home

## Display

- 8.8" IPS/LTPS, 2560x1600 (16:10), 275 PPI
- 144Hz refresh rate
- 500 nits brightness
- Touchscreen

## Processor & Graphics

- AMD Ryzen 7 8840U (Zen 4): 8 cores / 16 threads, up to 5.10 GHz, 28W TDP
- AMD Radeon 780M: 12 cores, up to 2700 MHz, shared memory

## Memory & Storage

- 16GB / 32GB / 64GB LPDDR5X 7500 MT/s
- 512GB / 1TB / 2TB M.2 2230/2242 PCIe 4.0 NVMe SSD (swappable)
- 1x MicroSD card slot (up to 2TB)

## Ports & I/O

- 2x USB 4.0 Type-C
- 1x USB-A 3.2
- 1x OCuLink (native)
- 1x MicroSD card slot
- 1x 3.5mm headphone jack
- Dual speakers by Harman
- Wi-Fi 6E, Bluetooth 5.2

## Battery

- 65.02 Wh Li-Po
- 100W GaN charger (50% in 30 minutes)
- Hades at 5W: 340 minutes

## Configurations

- 16GB LPDDR5X / 512GB NVMe PCIe 4.0 2230
- 16GB LPDDR5X / 1TB NVMe PCIe 4.0 2230
- 32GB LPDDR5X / 1TB NVMe PCIe 4.0 2230
- 32GB LPDDR5X / 2TB NVMe PCIe 4.0 2230
- 64GB LPDDR5X / 2TB NVMe PCIe 4.0 2230

## Availability

Out of stock at DROIX. Available new and refurbished (Class A).
