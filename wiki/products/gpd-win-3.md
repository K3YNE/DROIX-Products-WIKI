# GPD WIN 3

The GPD WIN 3 is [[gpd|GPD]]'s Intel Tiger Lake-era Windows handheld gaming PC. It features a slide-up display revealing a physical keyboard underneath, Thunderbolt 4 connectivity, and Intel Iris Xe graphics. Sold by [[droix|DROIX]] with a 2-year warranty. Now discontinued.

## Specs

| Spec | Detail |
|------|--------|
| CPU | Intel Core i5-1135G7 / i7-1165G7 / i7-1195G7 |
| GPU | Intel Iris Xe Graphics |
| RAM | 16 GB LPDDR4x @ 4266 MHz (dual-channel) |
| Storage | Up to 1 TB PCIe NVMe SSD |
| Display | 5.5" touchscreen |
| Connectivity | Wi-Fi 6, Bluetooth 5.1 |
| Ports | Thunderbolt 4 |
| In-box | USB Type-C cable, power adapter, user manual |

## Design & Build

- Sliding display reveals a physical QWERTY keyboard beneath.
- Compact handheld form factor with integrated gamepad controls.
- Predecessor to the [[gpd-win-4]], sharing the sliding keyboard concept.
