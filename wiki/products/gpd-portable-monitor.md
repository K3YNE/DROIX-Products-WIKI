# GPD 15.6" 4K Portable Monitor

[[gpd|GPD]] sells a range of 15.6" 4K portable monitors through GPDStore, available in three variants: Adobe RGB colour-accurate, stylus-compatible, and basic touchscreen. All share the same 15.6" UHD 4K resolution and USB/mini-HDMI connectivity. Sold by [[droix|DROIX]] with a 2-year warranty. All currently out of stock.

## Variants

| Variant | Key Feature | Touchscreen | Battery | In-box extras |
|---------|-------------|-------------|---------|---------------|
| Adobe RGB | 100% Adobe Color Gamut, 10-point capacitive touch, macOS touch support | Yes | 10,000 mAh built-in | Power plug, user manual |
| Stylus Support | Included stylus, foldable case stand, macOS touch support | Yes | — | Power plug, stylus, user manual |
| Touchscreen | Basic touch, compact (14.17 x 8.4 x 0.43"), 920g | Yes | — | Power plug, user manual |

## Shared Specs

- 15.6" UHD 4K resolution.
- USB-powered, supports mini-HDMI and USB Type-C input.
- macOS touchscreen compatibility on all models.
