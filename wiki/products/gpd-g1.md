# GPD G1

The GPD G1 is [[gpd|GPD]]'s external GPU docking station powered by an AMD Radeon RX 7600M XT. It connects via OCuLink (PCIe), USB 4, or Thunderbolt 3/4. Refreshed in 2024 with expanded device compatibility. This is a **different product** from the [[onexplayer-g1]], which is a dual-keyboard Windows handheld. Sold by [[droix|DROIX]] with a 2-year warranty.

## Specs

| Spec | Detail |
|------|--------|
| GPU | AMD Radeon RX 7600M XT ([[amd-rx-7600m-xt]]) |
| Connectivity | OCuLink (PCIe), USB 4 (up to 40 Gbps), Thunderbolt 3 & 4 |
| Compatible GPD devices (OCuLink) | [[gpd-win-4]] (2023+), [[gpd-win-max-2]] (2023+), [[gpd-win-mini]] |
| Compatible devices (2024 revision) | GPD WIN Max 2, WIN 4, WIN Mini, ROG Ally X (and others via USB4/TB) |
| In-box | Power adapter, USB Type-C cable, user manual |

## Models

- **GPD G1 (2023)** — Original release. OCuLink compatibility with WIN 4 2023, WIN Max 2 2023, WIN Mini.
- **GPD G1 (2024)** — Updated compatibility list; supports ROG Ally X and additional devices.

Both models use the same RX 7600M XT GPU.
