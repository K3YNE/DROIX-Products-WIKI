# DROIX NH8 USB Hub with NVMe

USB-C hub from [[droix|DROIX]] with built-in PCI-e 22*80 NVMe storage expansion, designed for handheld gaming PCs. Features 4K@60Hz HDMI output, Gigabit Ethernet, SD/MicroSD card slots (up to 5Gbps), 2x USB-C ports, 1x USB-A 3.0 port, and USB-C PD charging. Plug-and-play, no drivers required.

## Specs

| Spec | Value |
|------|-------|
| Input | 1x USB-C (attached cable) |
| Power Delivery | USB-C PD (up to 100W) |
| Video Out | 1x HDMI (4K@60Hz) |
| USB | 2x USB-C, 1x USB-A 3.0 |
| Ethernet | RJ45 10/100/1000 Mbps |
| Storage | PCI-e 22*80 NVMe slot (built-in enclosure) |
| Card Slots | SD + MicroSD (up to 5Gbps) |
