# GPD MicroPC

The original GPD MicroPC is a compact UMPC aimed at IT professionals and field technicians. It features an Intel Celeron N4120, Windows 10 Pro, RS-232 serial port, dual-band Wi-Fi, and Gigabit Ethernet. Predecessor to the [[gpd-micropc-2]]. Sold by [[droix|DROIX]] with a 2-year warranty. Now discontinued.

## Specs

| Spec | Detail |
|------|--------|
| CPU | Intel Celeron N4120 |
| OS | Windows 10 Pro |
| Ports | RS-232 serial port, 1 Gigabit RJ45 Ethernet |
| Connectivity | Dual-band Wi-Fi |
| In-box | Power adapter, USB Type-C cable, user manual |

## Design & Build

- Ultra-compact form factor with thumb-typable keyboard.
- **RS-232 serial port included** — a key differentiator vs the [[gpd-micropc-2]], which removed this port.
- Designed for network engineers, sysadmins, and field technicians.
