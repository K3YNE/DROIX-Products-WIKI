# ONEXPLAYER X1 AMD

10.95" 3-in-1 Windows tablet/handheld from [[onexplayer|ONEXPLAYER]] powered by AMD Ryzen 7 8840U with Radeon 780M. Features a 2.5K 120Hz LTPS display, detachable controllers, Hall effect joysticks, native OCuLink, Harman AudioEFX speakers, and fingerprint + face recognition. 65.02 Wh battery with 100W GaN charger. Out of stock at DROIX.

## Design & Build

- 10.95" LTPS 2.5K display
- 3-in-1 device: tablet, console, laptop (with detachable controllers and keyboard)
- Ergonomic detachable gaming controller with enhanced curvature design
- Hall effect joysticks with precise tactile triggers
- Fingerprint sensor (power button) + AI front camera (Windows Hello facial recognition)
- Active cooling: liquid bearing fan, pure aluminium heat sink fins, dual copper heat pipes
- Harman AudioEFX dual speakers
- Windows 11 Home

## Display

- 10.95" IPS/LTPS, 2560x1600 (16:10), 275 PPI
- 120Hz refresh rate
- 540 nits brightness
- Touchscreen

## Processor & Graphics

- AMD Ryzen 7 8840U (Zen 4): 8 cores / 16 threads, up to 5.10 GHz, 28W TDP
- AMD Radeon 780M: 12 cores, up to 2700 MHz, shared memory

## Memory & Storage

- 32GB or 64GB LPDDR5X 7500 MT/s
- 1TB / 2TB / 4TB M.2 2280 PCIe 4.0 NVMe SSD
- Read speeds up to 7114 MB/s, write up to 6501 MB/s
- 1x MicroSD card slot (up to 2TB)

## Ports & I/O

- 2x USB 4.0 Type-C
- 1x USB-A 3.2
- 1x OCuLink (native)
- 1x MicroSD 4.0 card slot
- 1x 3.5mm headphone jack
- Dual speakers by Harman
- Wi-Fi 6E, Bluetooth 5.2

## Battery

- 65.02 Wh Li-Po (~8 hours battery life)
- 100W GaN charger included

## Configurations

- 32GB LPDDR5X / 1TB NVMe PCIe 4.0 2280
- 32GB LPDDR5X / 2TB NVMe PCIe 4.0 2280
- 64GB LPDDR5X / 4TB NVMe PCIe 4.0 2280
