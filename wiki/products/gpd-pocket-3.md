# GPD Pocket 3

The GPD Pocket 3 is [[gpd|GPD]]'s 8" modular UMPC with Thunderbolt 4, fingerprint scanner, and swappable expansion modules (KVM, RS-232). Powered by an Intel Pentium Gold 7505 with up to 4 TB NVMe storage. Predecessor to the [[gpd-pocket-4]]. Sold by [[droix|DROIX]] with a 2-year warranty. Now discontinued.

## Specs

| Spec | Detail |
|------|--------|
| CPU | Intel Pentium Gold 7505 (up to 3.50 GHz, 4 cores / 8 threads) |
| RAM | 16 GB LPDDR4x @ 3733 MHz |
| Storage | Up to 4 TB PCIe NVMe SSD |
| Display | 8" touchscreen, fingerprint scanner |
| Ports | Thunderbolt 4, modular expansion slot |
| Connectivity | Wi-Fi, Bluetooth |
| Expansion | KVM module, RS-232 module (sold separately) |
| In-box | Power adapter, USB Type-C cable, user manual, GPD stylus |
