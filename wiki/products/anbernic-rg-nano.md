# Anbernic RG Nano

Ultra-compact keychain-sized retro gaming handheld from [[anbernic|Anbernic]]. ARM Cortex-A7 @ 1.2 GHz, 64 MB DDR2 RAM. Tiny 1.54" IPS display, 240x240. Aluminium alloy CNC case. No Wi-Fi or Bluetooth. Includes keychain attachment. Also functions as clock and music player.

## Specs

| Spec | Value |
|------|-------|
| CPU | ARM Cortex-A7 @ 1.2 GHz (boost 1.6 GHz) |
| RAM | 64 MB DDR2 |
| Storage | 64 GB MicroSD (expandable to 512 GB) |
| Display | 1.54" IPS, 240x240, 220 PPI, no touchscreen |
| Battery | 1050 mAh Li-Po (~2.5 hours) |
| Connectivity | None (no Wi-Fi, no Bluetooth) |
| Ports | USB-C, mono speaker, audio via USB-C adapter |
| OS | Linux |
| Weight | 75 g |
| Dimensions | 7.1 x 4.3 x 1.7 cm |
| Case | Aluminium alloy CNC |
