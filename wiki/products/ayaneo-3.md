# AYANEO 3

Windows handheld gaming PC from [[ayaneo|AYANEO]] offering two processor tiers and two display options. The top configuration pairs the [[amd-ryzen-ai-9-hx-370|AMD Ryzen AI 9 HX 370]] (12 cores / 24 threads, 15–54W) with a Radeon 890M (16 cores), while the base model uses an AMD Ryzen 7 8840U (8 cores / 16 threads, 15–30W) with Radeon 780M (12 cores). Display choices include a 7-inch OLED at 144 Hz / 800 nits / 110% DCI-P3 or a 7-inch LCD at 120 Hz / 500 nits / 100% sRGB with VRR FreeSync — both 1920x1080 16:9 315 PPI touch panels. The standout feature is a modular controller system developed in collaboration with GuliKit, offering 56 layout combinations including a touchpad module and a 6-button fighting game module. Dual-stage Hall triggers provide 7.5mm long and 3mm short travel. Connectivity includes dual USB4, OCuLink, and Wi-Fi 6E. Currently in stock at DROIX.

## Modular Controller System

The AYANEO 3 introduces a modular controller system with 56 possible layout combinations. Modules include a touchpad and a dedicated 6-button fighting game layout. The magnetic suspension motor is a collaboration with GuliKit. This system is unique among current Windows handhelds tracked in this wiki.
