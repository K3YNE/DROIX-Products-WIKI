# GPD P2 Max 2022

The GPD P2 Max 2022 is [[gpd|GPD]]'s ultra-compact 8.9" mini laptop with an Intel Pentium Silver N6000 processor, 16 GB RAM, fingerprint scanner, and touchscreen display. A productivity-focused UMPC. Sold by [[droix|DROIX]] with a 2-year warranty. Now discontinued.

## Specs

| Spec | Detail |
|------|--------|
| CPU | Intel Pentium Silver N6000 |
| RAM | 16 GB LPDDR4x @ 2933 MHz |
| Storage | 1 TB PCIe NVMe SSD |
| Display | 8.9" touchscreen, fingerprint scanner |
| OS | Compatible with Windows 11 |
| In-box | USB Type-C cable, power adapter, user manual |
