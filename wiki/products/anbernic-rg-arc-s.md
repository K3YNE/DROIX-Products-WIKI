# Anbernic RG ARC-S

Budget retro gaming handheld from [[anbernic|Anbernic]] with a SEGA Saturn controller-inspired design. RK3566 quad-core Cortex-A55 @ 1.8 GHz, Mali-G52 MP2 GPU. 4" IPS display, 640x480, 4:3 aspect ratio. Runs custom Linux OS. Wi-Fi 5, Bluetooth 4.2, mini HDMI output.

## Specs

| Spec | Value |
|------|-------|
| CPU | RockChip RK3566 Cortex-A55 quad-core @ 1.8 GHz |
| GPU | Mali-G52 MP2 |
| RAM | 1 GB LPDDR4 |
| Storage | 16 GB internal + 128/256 GB MicroSD |
| Display | 4" IPS, 640x480, 200 PPI, no touchscreen |
| Battery | 3500 mAh Li-Po (~6 hours) |
| Connectivity | Wi-Fi 5, Bluetooth 4.2 |
| Ports | 2x USB-C, 3.5mm jack, Mini HDMI |
| OS | Linux |
| Weight | 440 g |
| Dimensions | 8.3 x 4.8 x 1.6 inches |
