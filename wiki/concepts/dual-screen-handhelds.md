# Dual-Screen Handhelds

Dual-screen handhelds are a growing trend in portable gaming, enabling native dual-display emulation and secondary-screen utility.

## Android Devices

### [[ayn-thor|AYN Thor]]
- **Displays:** 6" + 3.92", both AMOLED
- **Weight:** 380g — very portable for a dual-screen device
- **Character:** Compact, pocketable, premium feel

### [[ayaneo-pocket-ds|AYANEO Pocket DS]]
- **Displays:** 7" OLED (top) + 5" LCD (bottom)
- **Weight:** 540g — larger and heavier
- **Character:** Bigger screens for more immersive dual-screen gaming

Announced but not yet reviewed by [[droix|DROIX]].

## Windows Devices

### [[ayaneo-flip-1s-ds|AYANEO Flip 1S DS]]
- **Displays:** 7" OLED (top) + 4.5" IPS (bottom)
- **Weight:** 655g — heaviest of the reviewed dual-screen devices
- **Processor:** AMD HX 370
- **Character:** Clamshell form factor, full Windows PC gaming with dual-screen capability

## Primary Use Cases

### Dual-Screen Emulation
The core appeal — running dual-screen console emulators across two physical displays:

- **Drastic** — Nintendo DS emulator
- **MelonDS** — Nintendo DS/3DS emulator
- **AHA** — alternative emulator supporting dual-screen output
- **Simu** — Wii U emulator, early stages on Android

Setup requires manual configuration to map each emulator screen to the correct physical display.

### Secondary Screen Utility
The bottom screen can serve as a companion display for:
- Game guides and walkthroughs
- Discord chat
- Game stats and maps
- Stream chat while gaming

## Reality Check

Most users will likely keep the second screen off for the majority of their gaming time. Running both displays impacts:

- **Performance** — rendering to two screens simultaneously uses additional GPU resources
- **Battery life** — two active panels drain faster

The second screen is best treated as a bonus feature for specific use cases rather than an always-on expectation.

## Thor vs Pocket DS

The comparison comes down to personal preference — there is no clear winner:

| | [[ayn-thor|AYN Thor]] | [[ayaneo-pocket-ds|AYANEO Pocket DS]] |
|---|---|---|
| Portability | Lighter (380g), more compact | Heavier (540g), larger |
| Top screen | 6" AMOLED | 7" OLED |
| Bottom screen | 3.92" AMOLED | 5" LCD |
| Panel consistency | Both AMOLED | Mixed (OLED + LCD) |
