# GPD Shared Driver Package V4.1.0

Shared driver package for AMD-based [[gpd|GPD]] handhelds (2024–2025 models with 8840U / AI 9 365 / AI 9 HX 370 APUs). Same package used by [[gpd-win-4]] (2023/2025), [[gpd-win-mini]] (2025), [[gpd-win-max-2]] (2023/2025), and [[gpd-duo]].

After extracting, double-click `AutoInstallDrivers.bat` for automatic installation.

The bundled AMD graphics driver may lag behind AMD's latest — users can manually update from [AMD's downloads portal](https://www.amd.com/en/support/download/drivers.html).
