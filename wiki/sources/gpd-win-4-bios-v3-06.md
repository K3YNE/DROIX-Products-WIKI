# GPD WIN 4 (6800U) — BIOS V3.06

Current BIOS for the original [[gpd-win-4]] (6800U). Only upgrade if experiencing one of the issues listed below.

**Fixes:** Volume keys becoming unresponsive after sleep on Linux; bottom USB-C port not displaying in v3.04; device crash under abnormal conditions. Adds screen firmware update interface.
