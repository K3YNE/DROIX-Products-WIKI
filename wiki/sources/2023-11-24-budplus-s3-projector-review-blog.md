## Summary

DROIX blog review of the [[budplus-s3-projector]]. Published 2023-11-24 on droix.net.

## Emulation Notes

- | **INPUTS/OUTPUT** | 1x HDMI input (HDMI Type-A)/ Audio, Video Input such as TV Stick / Laptop / DVD / PS3 / Sony PS4 /Xbox / TV box 1x AV /Audio, Video Input such as TV, Video devices 1x Audio Output port,to external Headphone/Audio amplifier, etc. 1x USB Type-A 5V,Power Video Input/Output |
- It is possible to install your own apps. You can do this through the web browser or download the APKs to a USB drive and install from there. We installed [RetroArch](https://www.retroarch.com/) and set up a few emulators to see the performance. We saw very good performance for all the 8-bit and 16-bit systems. When we get to PlayStation 1 performance we start to see some performance issues on games. You can also get PSP running, but again you will see some performance issues. Overall it is very impressive for this CPU and a nice bonus to have if you are a retro gamer.
