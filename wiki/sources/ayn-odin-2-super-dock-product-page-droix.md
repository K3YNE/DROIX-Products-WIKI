# AYN Odin 2 Super Dock — DROIX Product Page

Product listing from [[droix|DROIX]] for the [[ayn-odin-2|AYN Odin 2]] Super Dock docking station.

## Key Details

- **In stock:** No
- **Compatibility:** AYN Odin 2, AYN Odin 1 Base, AYN Odin 1 Pro (Odin 1 Lite: no video output via USB-C, other functions work)

## Features

- **Video Output:** HDMI 2.0, up to 4K @ 60 Hz
- **NVMe Support:** M.2 NVMe M-key and B+M-key, SATA B+M-key SSDs (2230, 2242/2260, 2280)
- **USB Ports:** 2x USB 3.0 Type-A + 2x USB 2.0 Type-A
- **Ethernet:** 1x RJ45 @ 1000 Mbps
- **Storage Expansion:** 1x M.2 SSD 22x80 slot

## Product Description

Docking station designed for the AYN Odin 2 that enables big-screen gaming via HDMI output, NVMe SSD storage expansion, wired Ethernet, and peripheral connectivity. Supports wireless controller connection for TV/monitor gaming sessions. Also compatible with the original AYN Odin 1 (Base and Pro models).
