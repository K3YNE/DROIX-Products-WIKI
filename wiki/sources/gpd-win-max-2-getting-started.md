Welcome to the getting started guide for the [[gpd-win-max-2|GPD WIN MAX 2 2024]]. The GPD WIN MAX 2 2024 is a cutting-edge handheld gaming PC that combines the power of a desktop computer with the portability of a gaming console. Featuring the AMD Ryzen 7 8840U processor and Radeon 780M graphics, this compact device packs a serious punch.

The WIN MAX 2 2024 is designed for gaming enthusiasts who want to enjoy their favourite PC games on the go, offering a 10.1-inch touchscreen display with a default 1200P resolution (supporting up to 1600P) and a clamshell design that includes a full keyboard.

## Inspect the GPD WIN MAX 2

When you first receive your GPD WIN MAX 2, we recommend performing an inspection to ensure everything is in working order:

* **Check the device case/shell:** Carefully examine the outer case for any cracks, dents, or other damage that may have occurred during shipping. Pay special attention to the clamshell design and hinge mechanism.
* **Test all keyboard keys:** Use https://keyboard-test.space/ to ensure all keyboard keys are registering correctly. The GPD WIN MAX 2 features a full-sized keyboard, so test each key thoroughly.
* **Test all buttons and joysticks:** Ensure all gaming controls, including buttons, joysticks, and triggers, are responsive and registering properly at https://hardwaretester.com/gamepad. The GPD WIN MAX 2 features Hall sensor sticks, which should provide precise and drift-free input.
* **Touchscreen functionality:** Test every part of the 10.1-inch touchscreen to ensure it responds correctly. Make sure you can interact with all corners and areas of the screen without any unresponsive zones.
* **Verify clamshell mechanism:** Test the clamshell design to ensure it opens and closes smoothly. Check that the screen holds its position at various angles.
* **Test USB ports:** Plug in common USB devices to ensure all ports are functioning properly. The GPD WIN MAX 2 features USB 4, USB 3.2 Gen 2 Type-C, and USB 3.2 Gen 2 Type-A ports.
* **Check HDMI port:** Connect an HDMI cable to the device and a monitor or TV to confirm the HDMI output is working correctly.
* **Verify audio output:** Test the speakers and 3.5mm audio port to ensure clear audio output.
* **Test OcuLink port:** If you have compatible hardware, verify the functionality of the OcuLink (SFF-8612) port.
* **Check SD card readers:** Test both the Micro SD and SD card readers to ensure they are functioning properly.
* **Test Wi-Fi and Bluetooth:** Ensure the Wi-Fi 6 and Bluetooth 5.2 connections are working correctly.
* **Test the built-in camera:** Run the Windows camera software and confirm the camera is working properly.

If you have any issues, DROIX / GPD Store offers full support. Head over to the [Contact Us](https://gpdstore.net/contact-us/) page and reach out.

## Update GPD WIN MAX 2 Windows and Drivers

### Updating Windows 11

1. Open the Start menu by clicking the Windows icon or pressing the Windows Key.
2. Select the Settings icon (gear symbol) or type "Settings" in the search bar and press RETURN (Enter).
3. In the Settings window, click on "Windows Update" in the left sidebar.
4. To check for available updates, click the "Check for updates" button.
5. If updates are available, Windows will begin downloading them automatically.
6. Once downloaded, you may need to restart your GPD WIN MAX 2 to complete the installation process.

**Note:** Your GPD WIN MAX 2 is set to automatically check for updates, but it is a good practice to manually check periodically. You can customise update settings by clicking "Advanced options" in the Windows Update menu.

### How to Update Drivers on the GPD WIN MAX 2

#### Windows Update

1. Open the Start menu and click on Settings (gear icon).
2. Select "Windows Update" from the left sidebar.
3. Click "Check for updates."

Windows will automatically download and install available driver updates along with other system updates.

#### Device Manager

For more specific driver updates:

1. Right-click the Start button and select "Device Manager."
2. Expand the category for the device you want to update.
3. Right-click on the device and select "Update driver."
4. Choose "Search automatically for drivers."

#### AMD Drivers Updates

Since the GPD WIN MAX 2 uses AMD processors, the AMD Auto-Detect driver updates will update the GPU and chipset related drivers.

1. Download and install the AMD Auto-Detect and Install Driver Updates from [here](https://www.amd.com/en/support/download/drivers.html).
2. Run the software and follow the prompts to choose what to update or install.

**Note:** Always download drivers from official sources to ensure system stability and security. While Windows Update covers most essential drivers, using the AMD tool can help keep AMD-specific components optimised for your GPD WIN MAX 2. See also [[gpd-amd-graphics-drivers|How to uninstall and install AMD graphics drivers]].
