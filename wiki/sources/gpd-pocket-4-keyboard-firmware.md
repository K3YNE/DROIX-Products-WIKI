# GPD Pocket 4 — Keyboard Firmware Update

Keyboard firmware update for the [[gpd-pocket-4]] that adds a keyboard backlight feature with three states: Off, On for 30 seconds then auto-off, and Always On. Toggle with Fn + Spacebar. Default state on first boot is "On for 30 seconds then auto-off".
