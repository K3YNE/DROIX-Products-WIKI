Welcome to our short article on a (potential) fix for the GPD WIN 3’s Rumble and Keyboard issues. In this article, we’ll go over a brief description of the issue, and what you can do to fix it.

## The GPD Win Rumble And Keyboard Issue?

Among the usual issues one will encounter with devices (for every one hundred devices, one unit might be a dud with a faulty battery etc.), some common issues we have observed with the few faulty GPD Win 3 we receive are the following:

* The device’s rumble becomes “stuck”. The GPD WIN 3 constantly rumbles.
* The GPD WIN 3 keyboard no longer works. It becomes non-responsive.

If your device is experiencing any of these issues, then there’s a possibility you can fix it! As it may be due to software-related problems, rather than strictly hardware.

## You Will Need

To install the update, you will only need two items.

* The [Update Files](https://droidbox.sharepoint.com/:u:/s/Purchasing/EcQuxq7WoSZJsZsrNxhUewkB6vwlgdcVSLPjnM68Os9fPQ?e=IopkBy)
* File Archival Software
  + Our recommendation is [7zip](https://www.7-zip.org/), although you can use just about anything.

With those downloaded, we can now move onto the installation process.

## How To Install

The installation for this fix is fortunately incredbily simple.

First, extract the update file from the .zip it came in.

Unzip the file via whatever method you wish.

Inside the .zip there will be two files. Run the one called **Hynitron\_Touch\_Update\_Tool(4).exe**.

…and that’s it!
