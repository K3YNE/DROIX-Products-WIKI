# GPD WIN 5 — BIOS v2.25

Current BIOS for the [[gpd-win-5]]. AMD updated the BIOS code; this is a completely new version. **No rollback to older BIOS versions after upgrading.**

## Install

In Windows, extract the archive (WinRAR password `123`), then double-click to execute. The device auto-restarts into the BIOS interface, performs the flash, and shuts down automatically. A manual restart follows.

**Device must be on mains power** during the upgrade.
