# GPD WIN 5 — Getting Started

First-party onboarding article from the GPD Store support hub. Guides new [[gpd-win-5|GPD WIN 5]] owners from unboxing through first boot, Windows update, driver update, and baseline optimisation. Published **2025-10-06** by Dave.

## What this article covers

- **Unboxing inspection checklist** — visual inspection of the casing; gamepad button/joystick test via `hardwaretester.com/gamepad`; touchscreen corner-and-edge test; USB port function test; audio output (speakers + 3.5 mm jack); battery-connector visual check before first attach.
- **Windows 11 update** — step-by-step through `Settings → Windows Update → Check for updates`, including reboot finalisation and a recommendation to run a manual check periodically even with auto-update enabled.
- **Driver update — Windows Update method** — the preferred path for most drivers, since Windows Update bundles driver updates alongside system updates.
- **Driver update — Device Manager method** — for targeted per-device updates (right-click Start → Device Manager → expand category → right-click device → Update driver).
- **Related:** first-party BIOS and driver releases covered separately — see [[gpd-win-5-bios-v2-25]] and [[gpd-win-5-drivers-2025-10-30]]. Power-management tuning is handled in [[gpd-motionassistant-v1-2-0-9|MotionAssistant]].

## When to use this page

Quote this article as the canonical first-party onboarding reference for:

- Customer-support responses where the issue is likely solvable by a Windows Update / driver refresh.
- Unboxing-problem triage (e.g., non-responsive button, dead pixel, USB port fault) — the article's inspection checklist defines the "report within X days" bar.
- Any "first things to do with my new WIN 5" knowledge-base query.
