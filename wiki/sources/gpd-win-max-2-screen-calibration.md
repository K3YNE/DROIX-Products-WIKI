# GPD WIN Max 2 — Screen Color Calibration File

ICM color profiles for the [[gpd-win-max-2]] display (Windows only). Applies to all generations. Available in two variants:

- **D65** (6500K): Simulates average daylight, cool-toned with higher blue light proportion.
- **D70** (~7000K): Cooler, bluish-white calibration for specialized high color temperature scenarios.

**Installation:** Copy the .icm file to `C:\Windows\System32\spool\drivers\color`, open Color Management, select the target monitor, check "Use my settings for this device", click Add, select the profile, then in the Advanced tab click "Change system defaults" and "Reload current calibration".
